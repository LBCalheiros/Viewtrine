# Viewtrine
Landing Page para plataforma de recrutamento especializada no setor da moda.

# Objetivo

- Simular uma aplicação próxima de um cenário real de mercado, priorizando:

- Estrutura HTML semântica

- CSS organizado e escalável

- Componentização visual

- Responsividade mobile-first

- JavaScript para interações específicas (menu mobile e estados ativos)

# Tecnologias utilizadas

- HTML5

- CSS3

- JavaScript (ES6)

(Sem uso de frameworks ou bibliotecas externas)

# Responsividade

A interface foi projetada para se adaptar a diferentes resoluções:

- Desktop (layout completo em grid)

- Tablet (reorganização estrutural)

- Mobile (menu hamburguer e scroll horizontal controlado)

Breakpoints principais:

- 1440px

- 1024px

- 768px

- 560px

# Funcionalidades

- Variação de layout para Candidato e Recrutador

- Menu mobile com animação

- Seções estruturadas com Grid e Flexbox

- Componentes reutilizáveis (botões, tipografia e cards)

- Uso de data-attributes para variações visuais dinâmicas

# Decisões técnicas

- Layout híbrido utilizando CSS Grid e Flexbox

- Classes utilitárias para controle tipográfico

- Estrutura modular de CSS via @import

- Controle de overflow horizontal em dispositivos móveis

- Organização sem dependência de frameworks para manter controle total da estrutura

# Possíveis evoluções

- Refatoração para arquitetura baseada em componentes

- Conversão para React (foi feito no projeto final)
